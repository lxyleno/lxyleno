2018年度总结

发表于 2019-02-03 | 分类于 生活 | 阅读次数：

序言

明天就是除夕了，想来一直都没有对2018年做个总结。
趁着农历新年将至，便做一份迟来的年度总结
也算是对逝去的一年有个交代。

其实，我这个人对于时间的流逝没什么概念。
往往到了某一个时间点，再回望过去，才发现我经历了这么多。
现在亦是如此。似乎我一路走来，只在一瞬之间。
我翻阅着手机里的一张一张相片，脑海中的回忆交织翻涌。

回忆2018

第一次独自出游（观泰山日出）

我犹记得那是从晚上9点开始历经5个小时，不停地攀爬，终于在凌晨两点登顶泰山。
休息了几个小时后，顶着凌冽的山风，看完了日出。其实当时我并没感受到所谓日出的磅礴之气，
脑海中只回荡着一个字——冷。那是一种冻到骨子里的冷。我也终于体会到所谓刺骨的寒风到底是什么样的。
不过也不尽是些痛苦的回忆，下山途中细细品味群山峻岭，倒也体会到了一些一览众山小的意境。（人少一点就更好了）

第一次与同学去看演唱会（五月天-石家庄）

虽然之前也看过一场演唱会，但那时完全没什么印象。这次和铎一起去，也可以说是第一次投入一场演唱会。
我可能是太激动了，一直在跟着唱。导致演唱会结束回酒店后，铎跟我吐槽我唱的声音又大又难听。
现在想想好尴尬的，哈哈哈。

对我来说或许是大学生涯最重要的一件事——转专业

其实之前我一直在焦虑到底转不转专业。
在原来专业难得能碰上志同道合又三观极正的铎，转专业之后我没的关系会不会疏远
未来我的同学会是什么样，未来我的室友会是什么样我学习能跟上吗。
类似于这样的问题一直在我脑海中回荡，不过现在看来一切都是我多虑了。
我很幸运和铎的关系没淡，也很幸运能碰上一群可爱的舍友。
这些日子里，让我印象最深的便是我早上起来毛手毛脚地把阿泽的保温壶踢翻了，
滚烫的开水浸透了他的整只脚。我一时手足无措。可他丝毫没有怪我的意思，
依然对我没心没肺的说着骚话。现在想起来内疚之余，我的心中还多了一点自惭形秽。
想来若是我被这样烫了脚，我可能不会这么宽宏大量。
可以说，阿泽几乎满足了我对青春少年的所有憧憬，高大帅气阳光正直，还会跳街舞。
如果我是女生，我也应该会喜欢这样的男孩子（过了过了）。
还有另外的六位舍友也是各有各的特点，但他们都一样可爱，
都是我喜欢的模样。

2018,这是我能想到最值得纪念的三件事。
2019，开始了…

2019小心愿

1.努力学习，不挂科

2.努力学英语，必须过四级

3.能遇到一个令我心动的女生

4.稍微减点肥

5.至少一次旅游

6.学会简单的穿搭

结语

最近看了一些关于宇宙的纪录片，我越是看越是感到深深的恐惧。
在宇宙中，地球甚至不及沧海一粟，而我也不过是地球上的众多碳基生物之一。
我的存在之于这个世界基本无意义，而我的人生就只有短短几十年。
所以，真的 想做什么就去做吧。
别太在意别人的看法，为自己而活。
人生不长，好好把握。

-年度总结
