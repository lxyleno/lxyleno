2018年度总结

发表于 2019-02-03 | 更新于 2019-02-04 | 分类于 生活

序言

明天就是除夕了，想来一直都没有对2018年做个总结。
趁着农历新年将至，便做一份迟来的年度总结
也算是对逝去的一年有个交代。

其实，我这个人对于时间的流逝没什么概念。
往往到了某一个时间点，再回望过去，才发现我经历了这么多事。
现在亦是如此。似乎我一路走来，只是一瞬之间的事。
我翻阅着手机里的一张一张相片，脑海中的回忆翻涌着。

回忆2018

第一次独自出游（观泰山日出）

我犹记得那是从晚上9点开始历经5个小时，不停地攀爬，终于在凌晨两点登顶泰山。
休息了几个小时后，顶着凌冽的山风，看完了日出。其实当时我并没感受到所谓日出的磅礴之气，
脑海中只回荡着一个字——冷。那是一种冻到骨子里的冷。我也终于体会到所谓刺骨的寒风到底是什么样的。
不过也不尽是些痛苦的回忆，下山途中细细品味群山峻岭，道也体会到了一些一览众山小的意境。（人少一点就更好了）

第一次与同学去看演唱会（五月天-石家庄）

虽然之前也看过一场演唱会，但那时完全没什么印象。这次和铎一起去，也可以说是第一次投入一场演唱会。
我可能是太激动了，一直在跟着唱。导致演唱会结束回酒店后，铎跟我吐槽我唱的声音又大又难听。
现在想想也挺尴尬的，哈哈哈。

还有对我来说也或许是大学生涯最重要的一件事——转专业

其实之前我一直在焦虑到底转不转专业。
在原来专业难得能碰上志同道合又三观极正的铎，转专业之后我没的关系会不会疏远
未来我的同学会是什么样，未来我的室友会是什么样我学习能跟上吗。
类似于这样的问题一直在我脑海中回荡，不过现在看来一切都是我多虑了。
我很幸运和铎的关系没淡，也很幸运能碰上一群可爱的舍友。
这些日子里，让我印象最深的便是我早上起来毛手毛脚地把阿泽的保温壶踢翻了，
滚烫的开水浸透了他的整只脚。我一时手足无措。可他丝毫没有怪我的意思，
依然对我没心没肺的说着骚话。现在想起来内疚之余，我的心中还多了一点自惭形秽。
想来若是我被这样烫了脚，我可能不会这么宽宏大量。
可以说，阿泽满足了我对青春少年的所有憧憬，高大帅气阳光正直，还会跳街舞。
如果我是女生，我也会喜欢这样的男孩子（过了过了）。
还有另外的六位舍友也是各有各的特点，但唯一不变的是他们都很可爱，
都是我喜欢的样子（别想多，就是单纯的钦慕）

2018,我经历了不少的事。但这是我能想到最值得纪念的三件事。
希望我的2019年能过得更加有意义。

2019小目标

1.努力学习，不挂科，最好拿奖学金

2.英语四级（再不考就凉了）

3.争取有个女朋友（毕竟20岁的小青年，还是对爱情充满了憧憬）

4.稍微减点肥（嘻嘻）
